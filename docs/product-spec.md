# Product Spec

## Product Name

AI Job Fit Analyzer

## Target User

Career changers applying for Sales Operations, Sales Support, RevOps, Business Analyst, or Operations-related roles.

## User Input

The user provides:

1. Job description
2. Resume summary

## System Output

The system generates:

1. Core job requirements
2. Resume match analysis
3. Partial match analysis
4. Skill and experience gaps
5. Resume rewrite suggestions
6. Interview preparation questions

## MVP Scope

The MVP does not scrape job sites automatically.  
The user manually pastes the job description and resume summary.

## Out of Scope

- Automatic 104 scraping
- Login-required job platform access
- Real-time job recommendation
- Full ATS scoring
- Paid API integration
